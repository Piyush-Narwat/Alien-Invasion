<h1>Alien Invasion</h1>
<body>
  <p>This is a simple 2D game. It is created using python and the pygame package. Here you can start the 
  game either by clicking on the play button or by the help of key 'p'.</p>
  <p>In this game there are some aliens with move sideways and come down a little as any one of them touches the 
  edge of the screen. The player has three lives as shown with the help of images of ships on the upper right 
  corner of the window. The ship in the middle of the bottom fires bullets (5 at a time) to shoot the aliens.
  If any of the alien touches the ship or the ground, the player losses a life and if all the three lives are gone,
  the game is over. The difficulty of the game increases with the level.</p>
  <p>It stores the high score permanently in a different file using the json module. With every level the speed of the ship, bullets, and aliens
  increases.</p>
  <p>Enter 'q' to quit the game any time you want.</p>
</body>
